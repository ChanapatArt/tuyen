# Project Tuyen (Smart Fridge App) 🥦🍗

ระบบตู้เย็นอัจฉริยะ ช่วยคิดเมนูอาหารจากวัตถุดิบที่มี และคำนวณโภชนาการ

## Structure
* `mobile_app/`: Source code for Flutter Application
* `backend_api/`: Python Backend (FastAPI) & Data Science Models
* `cloud_infra/`: Docker & Deployment configurations

## Getting Started for Data Sci Team
1. Navigate to `backend_api`
2. Install dependencies: `pip install -r requirements.txt`
3. Run API: `uvicorn app.main:app --reload`
